# Simulador Oceania React

Projeto React/Vite gerado a partir do HTML original do simulador CCCPM.

## Organização

```txt
src/
  App.jsx
  components/
    Simulador.jsx
    Capacidade.jsx
    Projecao.jsx
    Comparativo.jsx
    Metodologia.jsx
    UnitGrid.jsx
    ChartCanvas.jsx
  data/
    units.js
    financialTables.js
  utils/
    financial.js
    formatters.js
  styles.css
```

## Rodar localmente

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```
